# Abhishek Kumar | Portfolio Website 🚀

A modern, highly responsive, and interactive personal portfolio website designed to showcase my projects, skills, and experience. Built entirely without external frameworks for lightning-fast performance, it features a custom "Golden Hour & Deep Teal" dark mode aesthetic inspired by custom artwork.

## ✨ Features

- **Modern UI/UX**: Deep dark mode with glassmorphism effects, dynamic glow shadows, and a split-screen responsive hero section.
- **Micro-Animations**: Custom scroll-triggered fade-in animations built entirely with the native `Intersection Observer API`.
- **Fully Responsive**: Adapts seamlessly to all screen sizes (Desktop, Tablet, Mobile) with an interactive mobile hamburger menu.
- **Serverless Contact Form**: Fully functional contact form integrated with [FormSubmit](https://formsubmit.co/), securely routing messages straight to the inbox without needing a backend server.
- **Zero Dependencies**: Built with pure Vanilla web technologies meaning 0 installation time and instantly deployable anywhere.

## 🛠️ Tech Stack

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom variables, Grid/Flexbox layouts, keyframe animations, and modern glassmorphism (backdrop-filter).
- **JavaScript (ES6+)**: DOM manipulation and scroll observation.
- **Fonts & Icons**: Google Fonts ('Outfit' & 'Inter') and Font Awesome 6.

## 🚀 Running Locally

Because this project is built entirely with static files, there is zero setup required.

1. Clone the repository:
   ```bash
   git clone https://github.com/Abhikrr2/portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in any modern browser, or use a local server like VS Code's **Live Server** extension for real-time reloading.

## 🌐 Deployment

This project is optimized to be deployed instantly on platforms like **GitHub Pages**, **Netlify**, **Vercel**, or **Render**. Simply connect the repository to your hosting provider of choice and set the root directory as the deployment source.
